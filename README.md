# 関東有名ケーキ店・カフェマップ

関東のケーキ店・カフェ候補を地図で見るための静的HTMLツールです。

## 開き方

`index.html` をブラウザで開きます。

## 主な機能

- ケーキ/カフェの表示切り替え
- エリア、タイプ、訪問状況での絞り込み
- 店名・駅・特徴の検索
- 「行った」フラグの保存
- データ更新用のJSON/CSV取り込み
- 閉店チェック用リンク生成

## 注意

- 地図表示にはインターネット接続が必要です。
- 「行った」フラグと追加データはブラウザのlocalStorageに保存されます。
- 営業時間、定休日、閉店状況は訪問前に公式情報で確認してください。
